hive on spark 框架搭建

发表于 2019-05-02 | 更新于: 2019-05-15 | 分类于 大数据处理 ， hadoop | 阅读次数: |

看啥摘要，进来看呀

hive on spark对hive和spark的版本要求比较严格。之前调试踩了各种坑，最后屈服，使用了hadoop 2.6，spark 2.0.0，hive 2.3.4

调度统一使用YARN

官方链接
https://cwiki.apache.org/confluence/display/Hive/Hive+on+Spark%3A+Getting+Started

1. 安装Java

2. 部署Mysql（用来放hive的元数据）

3. 安装maven

4. 安装hadoop（伪分布式）

下载hadoop对应版本：全版本hadoop下载链接

解压后，在 ~/.bashrc 文件中配置

然后
$ hadoop version

应该会看到

为了使用Java开发Hadoop程序，必须用java在系统中的位置替换JAVA_HOME值并重新设置hadoop-env.sh文件的java环境变量
export JAVA_HOME={}

更改配置文件 官方链接

hadoop的配置文件都放在了$HADOOP_HOME/etc/hadoop下
$ cd $HADOOP_HOME/etc/hadoop

以下是必须编辑配置Hadoop的文件列表。

core-site.xml
core-site.xml文件中包含如读/写缓冲器用于Hadoop的实例的端口号的信息，分配给文件系统存储，用于存储所述数据存储器的限制和大小。

hdfs-site.xml

hdfs的配置文件，至少需要设置副本数dfs.replication，NameNode路径信息(fsimage)，以及本地文件系统的数据节点的路径

yarn-site.xml

需要配置NodeManager上的附属服务mapreduce_shuffle，才能运行MR任务

mapred-site.xml

MR的配置文件，需要指定运行在yarn上

$ cp mapred-site.xml.template mapred-site.xml

配置完成

验证Hadoop安装

第1步：名称节点设置

第一次运行之前，先在NameNode上初始格式化
hdfs namenode -format

第2步：起hdfs

$ start-dfs.sh

启动成功的话，jps里面会有DataNode、NameNode、SecondaryNameNode

第3步：起yarn

$ start-yarn.sh

启动成功的话，jps里面会有ResourceManager、NodeManager

第4步：UI访问hadoop

Hadoop默认端口号为50070, 启动成功的话
http://localhost:50070/
应该可以成功连上

第5步：验证所有应用程序的集群

访问集群中的所有应用程序的默认端口号为8088。（yarn调度的这个）
http://localhost:8088/
成功连上差不多就是搞定了

5. 安装spark (yarn调度)

spark必须选择spark源码进行编译(为了不带hive，防止版本冲突)，而且还要严格的遵守hive提供的支持版本进行下载（否则后果很可能是放弃）
https://archive.apache.org/dist/spark/spark-2.0.0/

解压后编译（要花很多时间，下载各种依赖）
$ ./make-distribution.sh --name "hadoop2-without-hive" --tgz "-Pyarn,hadoop-provided,hadoop-2.6,parquet-provided"

安装好spark再配一下SPARK_HOME

更改配置文件

只需要配置一个文件/conf/spark-env.sh就行了（因为yarn调度，所以spark就省事了）

spark-env.sh

提交spark任务的时候，master指定yarn就行了
$ spark-submit --master yarn {}

然后在yarn的UI界面可以看到对应的spark任务
http://localhost:8088/

6. 安装hive

hive我用的是2.3.4，理论上来讲2.3.*应该都可以
下载链接：
http://apache.website-solution.net/hive/hive-2.3.4/apache-hive-2.3.4-bin.tar.gz

解压后配一下HIVE_HOME

更改配置文件

hive-env.sh

hive-site.xml

$ cp hive-default.xml.template hive-site.xml
